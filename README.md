# 🎓 The Complete HTML Course: Multi-Page Platform

This is a fully responsive educational website for learning HTML5 fundamentals.

## 🔗 Live Demo
🚀 **Explore the live project here:** [https://fatitab-dev.github.io/HTML-Course-Platform/index.html](https://fatitab-dev.github.io/HTML-Course-Platform/index.html)

## 📄 Project Pages
* **Home Page**: `index.html` (Course introduction & syllabus)
* **Login Interface**: `login.html`
* **Registration**: `sing in.html`
* **Support**: `contact.html`

## 🚀 Technical Highlights
* **Responsive Architecture**: Fully optimized for mobile, tablet, and desktop using CSS Media Queries.
* **Advanced Layouts**: Implemented with **CSS Flexbox** for perfect alignment across all four pages.
* **Unified Styling**: A central `style.css` file manages the royal purple theme (#350224), rounded borders, and soft shadows globally.
* **Semantic HTML5**: Used to ensure accessibility and a clean document structure across the entire site.

## 🛠 Tech Stack
* **HTML5**: Multi-page structure and semantic content.
* **CSS3**: Custom UI components, responsiveness, and layout management.
